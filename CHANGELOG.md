# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-07

### Added

- Floating widget with playback controls (play/pause, next, previous)
- Now playing display with track title, artist, and artwork
- Scrolling marquee for long text
- Favorite/unfavorite tracks
- Settings panel with opacity slider and color picker
- Dynamic text color based on background luminance
- Launch on login option via LaunchAgent
- Horizontal window resizing
- Quit confirmation dialog
- App bundle with custom icon
- Makefile for building and distribution
- GitHub Actions CI/CD workflows
